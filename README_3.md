##  Overview of the Project
This is a simple console-based Hospital Management System made using Python.  
The project allows users to *Add, View, Update, and Delete (CRUD)* patient records.  
It is beginner-friendly and demonstrates the basic use of loops, lists, dictionaries,  
and CRUD operations in Python.

##  Features
- Add new patient details  
- View all registered patients  
- Update existing patient information  
- Delete a patient record using ID  
- Auto-increment patient ID  
- Simple console-based interface  
- Easy to run for beginners

##  Steps to Install & Run the Project
1. Install Python 3.x  
2. Download or clone the project folder  
3. Open the folder in VS Code  
4. Run the file using:


python hospital.py


5. Follow the menu on the screen to perform CRUD operations
